API Documentation

RESTful API for Healthcare Readmission Prediction

🚀 Quick Start

Make your first API call in seconds:

curl -X POST http://127.0.0.1:5000/api/predict \ -H "Content-Type: application/json" \ -d '{"age": 65, "time_in_hospital": 4, "num_lab_procedures": 40, "num_medications": 10, "number_outpatient": 0, "number_emergency": 1, "number_inpatient": 0}'

Endpoints

GET /

Returns the web interface home page with the interactive prediction form. This is the main entry point for browser-based access.

// Response: HTML page with prediction form Content-Type: text/html

POST /predict

Form-based prediction endpoint. Submit patient data via HTML form fields and receive results as a rendered HTML page with SHAP explanations.

// Form Data (application/x-www-form-urlencoded) age=65 time_in_hospital=4 num_lab_procedures=40 num_medications=10 number_outpatient=0 number_emergency=1 number_inpatient=0
// Response: Rendered HTML result page

POST /api/predict

RESTful JSON API for programmatic access. Send patient data as JSON and receive prediction with confidence score and SHAP explanations.

Request:

POST /api/predict Content-Type: application/json { "age": 65, "time_in_hospital": 4, "num_lab_procedures": 40, "num_medications": 10, "number_outpatient": 0, "number_emergency": 1, "number_inpatient": 0 }

Success Response (200 OK):

{ "prediction": 1, "prediction_text": "Readmitted", "confidence": 0.7234, "contributing_factors": [ {"feature": "Emergency Visits", "impact": 0.0821}, {"feature": "Patient Age", "impact": 0.0534}, {"feature": "Time in Hospital", "impact": 0.0312} ] }

Error Response (400 Bad Request):

{ "error": "Missing required field: age" }

GET /history

View the last 10 predictions made during the current session. History is stored in session storage and clears on browser restart.

// Response: HTML page with prediction history table Content-Type: text/html

Input Validation

All numeric fields are validated for range. Requests with invalid data will receive a 400 Bad Request response.

Field Type Min Max Description
age integer 0 120 Patient age in years
time_in_hospital integer 0 365 Days spent in hospital
num_lab_procedures integer 0 500 Number of lab procedures
num_medications integer 0 200 Number of medications
number_outpatient integer 0 100 Outpatient visits (past year)
number_emergency integer 0 50 Emergency visits (past year)
number_inpatient integer 0 50 Inpatient visits (past year)

Integration Examples

Python

import requests url = "http://127.0.0.1:5000/api/predict" data = { "age": 65, "time_in_hospital": 4, "num_lab_procedures": 40, "num_medications": 10, "number_outpatient": 0, "number_emergency": 1, "number_inpatient": 0 } response = requests.post(url, json=data) result = response.json() print(f"Prediction: {result['prediction_text']}") print(f"Confidence: {result['confidence']}")

JavaScript (Fetch)

const response = await fetch('http://127.0.0.1:5000/api/predict', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ age: 65, time_in_hospital: 4, num_lab_procedures: 40, num_medications: 10, number_outpatient: 0, number_emergency: 1, number_inpatient: 0 }) }); const result = await response.json(); console.log('Prediction:', result.prediction_text);
